
# Health Vault

HealthVault is a secure health records backend built using Spring Boot and PostgreSQL that allows patients to upload, store, and share medical files with doctors. It uses a hybrid encryption model (AES + RSA) to ensure that files remain private and accessible only to authorized users, following a zero-knowledge approach. The system includes JWT-based authentication, role-based access control, secure record sharing, and paginated APIs, along with Swagger documentation for easy testing and integration.
